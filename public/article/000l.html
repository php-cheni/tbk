linux系统下一键安装lnmp

以前学习过的linux centos，很久都没有去用，慢慢都忘光了，最近又开始有接触了。很多东西都不会，重新开始学习。

LNMP一键安装包是什么?

LNMP一键安装包是一个用Linux Shell编写的可以为CentOS/RHEL/Fedora/Aliyun/Amazon、Debian/Ubuntu/Raspbian/Deepin/Mint Linux VPS或独立主机安装LNMP(Nginx/MySQL/PHP)、LNMPA(Nginx/MySQL/PHP/Apache)、LAMP(Apache/MySQL/PHP)生产环境的Shell程序。


1.lnmp下载

//下载 
wget -c http://soft.vpser.net/lnmp/lnmp1.5.tar.gz



2.解压lnmp压缩包

//解压
tar -zxvf lnmp1.5.tar.gz


3.安装lnmp

//安装 
cd lnmp1.5 
./install.sh


接下来就是一些安装选项

首先是选择mysql的版本 输入数字选择对应的版本 回车选择默认选项（默认5.5的版本）

然后输入mysql的密码

innodb选择

接下来是选择PHP的版本（默认5.6版本）

继续回车，选中默认不安装

Please any key to install … or Press Ctrl+c to cancel

点击任意键继续安装，然后就是漫长的安装过程（大概要15分钟）。可以先去做点别的。

Install lnmp V1.5 completed! enjoy it.

出现了这个就安装完成了，按Ctrl+c 完成安装，可以访问localhost试试


最近linux接触比较多，会花时间来深入学习，希望大家多关注！
