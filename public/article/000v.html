原生PHP使用 PHPMAILER发送邮件 非常简单

使用PHPMAILER 无论是使用框架还是原生都可以非常简单的实现发送邮件

首先就是要找一个PHPMAILER,网上有很多版本大家可以选择一个自己适用的就行。我把我使用的分享出来。

PHPMAILER1.80
链接：https://pan.baidu.com/s/1plH8ME9rGkHkf6-qmFaK4Q 密码：5xj4

一般只用到这三个文件，可以直接单独拿出来

引入之后调用

require '/../include/class.phpmailer.php'; 
require '/../include/class.smtp.php'; 
function send_mail($to,$fromname,$title,$content){
    try { 
        $mail = new PHPMailer(true); 
        $mail->IsSMTP(); 
        $mail->CharSet='UTF-8'; //设置邮件的字符编码，这很重要，不然中文乱码 
        $mail->SMTPAuth = true; //开启认证 
        $mail->Port = 25; //端口请保持默认
        $mail->Host = "smtp.163.com"; //使用QQ邮箱发送
        $mail->Username = "cxk_king@163.com"; //这个可以替换成自己的邮箱
        $mail->Password = "Cxk654080169"; //注意 这里是写smtp的授权码 写的不是QQ密码，此授权码不可用
        //$mail->IsSendmail(); //如果没有sendmail组件就注释掉，否则出现“Could not execute: /var/qmail/bin/sendmail ”的错误提示 
        $mail->AddReplyTo("cxk_king@163.com","mckee");//回复地址 
        $mail->From = "cxk_king@163.com"; 
        $mail->FromName = $fromname; 
        $to = $to; 
        $mail->AddAddress($to); 
        $mail->Subject = $title; 
        $mail->Body = $content;
        $mail->AltBody = "To view the message, please use an HTML compatible email viewer!"; //当邮件不支持html时备用显示，可以省略 
        $mail->WordWrap = 80; // 设置每行字符串的长度 
        //$mail->AddAttachment("f:/test.png"); //可以添加附件 
        $mail->IsHTML(true); 
        $mail->Send(); 
        echo "发送成功！";
    } catch (phpmailerException $e) { 
        echo "邮件发送失败：".$e->errorMessage(); 
    } 
}
